Введение

О cypherGo

Менедежер тектовых файлов предназначенный для шифрования ваших документов, с возможностью загрузки больших объемов данных в систему.

О нас

Мы маленькая компания разработчиков работающих ради достижения своих целей.

Функционал

В данном разделе вы можете просмотреть весь список функциональных возможностей cypherGo!

Аунтефикация

Данная функция позволяет авторизироваться пользователю для дальнейшей работы с приложением. Также она позволяет пройти регистрацию для создания аккаунта.

Файлы

Функция “Файлы” позволяет добавлять, удалять/изменять или сохранять файлы.

Добавить

- Добавление файла с ПК. При добавлении с ПК пользователь выбирает файл находящийся на диске пользователя.

- Добавление файла в ручную. При добавлении файла вручную пользователь вводит название файла, который хочет создать, и добавляет его в приложение.

Редактировать в ручную

Данная функция позволяет по нажатию кнопки “Редактировать вручную” редактировать содержимое выбранного файла.

Название

Функция “Изменить название файла” позволяет менять имя выбранного файла.

Удалить

- Удалить все файлы. Данная функция позволяет удалить все файлы пользователя в приложении.

- Удалить выбранный файл. Данная функция позволяет переместить выбранный файл в корзину.

Сохранить

- Функция “Сохранить исходные данные” позволяет сохранить содержимое файл формата .txt на диск ПК.

- Функция “Сохранить зашифрованные данные” позволяет сохранить зашифрованные данные в файл формата .txt на диск ПК.

- Функция “Сохранить расшифрованные данные” позволяет сохранить расшифрованные данные в файл формата .txt на диск ПК.

Корзина

- Функция “Посмотреть содержимое”. Данная функция позволяет взаимодействовать с содержимым корзины.

- Функция “Очистить корзину”. Данная функция позволяет безвозвратно удалить содержимое корзины.

Восстановить

Функция “Восстановить файл” позволяет вернуть файл из корзины для дальнейшего взаимодействия с ним.

Интервал

Функция “Интервал” при нажатии правой кнопки мыши в корзине позволяет выбрать интервал автоотчищнеия корзины.

Справка

- Функция “Связь с разработчиками” перейти на аккаунт разработчиков для дальнейшей связи с ними. Также она позволят сохранить в буфер обмена почту разработчиков.

- Функция “Сайт приложения” переносит пользователя на официальный сайт приложения.

Соц Сети

Данная функция позволяет перейти на социальные сети разработчиков.

Возникла ошибка?

Форма позволяющая расписать свою проблему техническому администратору.

Связь с администрацией

Функция “Обратиться к администрации” позволяет написать администраторам приложения о возникшей ошибке.

Зашифровать

Данная функция позволяет открыть параметры шифрования.

Параметры

Данная форма позволяет выбрать параметры шифрования.

Сдвиг

Данная функция позволяет выбрать длину сдвига при шифровании Цезаря.

Алфавит

Данная функция позволяет выбрать режим шифрования:

- Зашифровать всё. Позволяет зашифровать весь файл.

- Зашифровать только русские буквы. позволяет зашифровать только русский алфавит.

- Зашифровать только английские буквы. позволяет зашифровать только английский алфавит.

- Зашифровать только символы. Данная функция позволяет зашифровать только символы.

Виды

Данная функция позволяет выбрать один вид шифрования из предоставленного списка всех доступных шифров.

Шифр Цезаря

Шифр Цезаря вид шифра подстановки, в котором каждый символ в открытом тексте заменяется символом находящимся на некотором постоянном числе позиций левее или правее него в алфавите.

Шифр Виженера

Шифр Виженера метод полиалфавитного шифрования буквенного текста с использованием ключевого слова. В шифре Цезаря каждая буква алфавита сдвигается на несколько позиций; например в шифре Цезаря при сдвиге +3, A стало бы D, B стало бы E и так далее. Шифр Виженера состоит из последовательности нескольких шифров Цезаря с различными значениями сдвига.

Панель администратора

Данная функция позволяет перейти в панель администратора. Данная функция доступна только администр... НИКОМУ

Пользователи

Данная функция позволяет просмотреть список администраторов, а также добавить нового. Также она позволяет удалять, блокировать и уведомлять пользователей.

Удалить

Данная функция позволяет удалить аккаунт пользователя из базы данных.

Заблокировать

Данная функция позволяет заблокировать выбранного пользователя. Аккаунт пользователя останется в базе данных.

Разбан

Данная функция позволяет разблокировать выбранного пользователя.

Уведомить

Данная функция позволяет уведомить пользователя.

- Функция “отправить письмо на почту”. Данная функция позволяет отправить письмо на почту выбранного пользователя.

- Функция “Отправить сообщение на телефон”. Данная функция позволяет отправить СМС сообщение выбранному пользователю.

Отчеты

Формирование отчетности.

Создание отчета

Данная функция позволяет создать отчет с выводом информации в шаблон документа формата .docx. В отчете будет содержаться информация о количестве зарегистрированных пользователей за определенный период времени.

Обратная связь

Данная функция позволят открыть форму с пользователями для просмотра всех оставленных жалоб.

Ответ пользователю

Данная функция позволяет выбрать пользователя для ответа на жалобу.
